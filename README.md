# shadcn-schema

[![NPM version](https://img.shields.io/npm/v/shadcn-schema?color=a1b858&label=)](https://www.npmjs.com/package/shadcn-schema)

A reusable schema for the [Shadcn registry](https://ui.shadcn.com/docs/registry/registry-json).

It exports the same API as [`shadcn/schema`](https://doc.deno.land/https://esm.sh/shadcn@4.11.0/schema), but is based on `zod` v4 instead of `zod` v3.
